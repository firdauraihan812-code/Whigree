# Whigree

Website server modern untuk top up Robux (demo) menggunakan Node.js + Express + vanilla frontend.

## Jalankan lokal

```bash
npm install
npm run start
```

Buka: `http://localhost:3000`

## Fitur

- Landing page modern + responsive.
- API paket Robux, metode pembayaran, health check.
- Form checkout dengan validasi backend.
- Simulasi riwayat order terbaru.
